# homebrew-tap

Homebrew tap for [Weaviate UI](https://github.com/harishkaparwan/weaviate-ui) — a workbench for inspecting and querying Weaviate databases.

## Install

```bash
brew tap harishkaparwan/tap
brew install --cask weaviate-ui
```

## Update

```bash
brew upgrade --cask weaviate-ui
```

## Uninstall

```bash
brew uninstall --cask weaviate-ui
# To also remove all app data:
brew uninstall --zap --cask weaviate-ui
```

## Requirements

- macOS on Apple Silicon (arm64)
- [Homebrew](https://brew.sh) installed

## Casks

| Cask | Description |
|------|-------------|
| `weaviate-ui` | Weaviate UI desktop workbench |
